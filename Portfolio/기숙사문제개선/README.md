## 설문 데이터 기반 기숙사 운영 문제 개선 (2025.09 ~ 2025.12)
- 2,700명 규모 기숙사의 자치회장으로 일하면서 대학 익명 커뮤니티(에브리타임)의 기숙사 게시판에 파편화된 불만을 보고, 한정된 예산으로 만족도를 높이려면 데이터 기반 객관적인 개선 우선순위가 필요하다고 판단함.
- 기존 기숙사 자치회는 게시판 불만을 생활원에 전달하는 역할에 그쳤고, 예산 등의 이유로 현실에 반영되기 어려웠음.
- 건의사항과 불만요인을 파악하기 위한 29개 문항의 정밀 설문을 설계하고 4주간 설문 데이터 수집.
- SPSS를 통해 Cronbach's α = 0.886을 도출하여 다문항 설문의 내적 일관성을 기술적으로 검증.
- 5단계 리커트 척도를 만족/불만족의 2단계 데이터로 재가공하여 응답을 명확하게 분류하고 각 성별 기숙사의 불만 1위(남성: Wi-Fi, 여성: 독서실)를 파악.
- Python을 활용해 서술형 응답 데이터를 텍스트 마이닝하고 워드클라우드로 시각화하여 '취사실 냉장고 도난 및 과점유, Wi-Fi 불안정' 문제가 최우선 개선이 필요함을 객관적으로 증명.
- 4가지 핵심 제안을 구체화한 보고서를 들고 기숙사 생활원장님과의 면담을 가져 데이터 기반의 설득을 성공적으로 함.
- 4가지 제안: (생활원 만족도조사 설문 문항 개선, 취사실 운영 체계 개선 방안, Wi-Fi 품질 개선, 기숙사 내 독서실 좌석 장기점유 관리 방안)

**성과: 제안 내용이 2026년도 운영계획에 공식 반영 및 교내 2025 정주대학 성과평가회 우수상 수상.**
