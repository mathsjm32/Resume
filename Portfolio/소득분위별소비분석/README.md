## 가계동향조사 기반 소득분위 계층별 소비구조 추세 분석 (2025.06 ~ ing)
- 활용 데이터: KOSIS(품목별/지출목적별 소비자물가지수), MDIS(가계동향조사) 2022년 1분기~2026년 1분기
- 분석 주제: 2022Q1~2026Q1(17개 분기) 동안 소득 5분위 계층별 소비구조(12대 항목 비중)가 어떻게 변했고, 계층 간 격차가 어떻게 움직였는지 추세 분석
- '소비자물가 상승률'이나 '가구당 평균 소비지출'과 같은 단일 평균 통계로는 소득분위마다 다르게 겪는 경기체감을 반영하지 못한다. 각 계층마다 어떤 품목의 지출과 소비를 줄이고 늘리는지, 경기 충격이 어떻게 다르게 영향을 주는지를 파악하고자 함.
