## 소득분위 계층별 소비구조 변화와 체감물가 부담의 분배적 분석 (2026.06 ~ 07)

- **활용 데이터:** 통계청 가계동향조사 가구 단위 분기 마이크로데이터(2019Q1–2026Q1, 29개 분기·191,096가구)와 KOSIS 소비자물가지수(2020=100). 두 자료의 분류 불일치(가계동향=신분류 / CPI=구분류)를 품목 단위로 재집계한 '분류 정합 디플레이터'로 해소해 명목·실질을 분리

- **분석 주제:** 소득 5분위별 12대 소비항목 비중(소비구조)의 변화, 계층 간 격차 추이, 그리고 물가 부담의 계층별 차등(체감물가 역진성)을 정량화 — "평균이 가린 계층별 소비 현실"을 분포로 분석

- **분석 방법:** 분기×소득5분위 가중집계 패널 구성, 계층별 라스파이레스/파셰/피셔 체감물가지수, 엥겔·슈바베계수, Working-Leser 지출 탄력성, 9종 ML 모델 비교 후 RandomForest 선정, SHAP 기반 설명가능성(XAI), 부트스트랩 신뢰구간·HAC 표준오차·확장윈도우 교차검증으로 불확실성 정량화

- **주요 결과:**
  - 소비구조: 저소득층은 필수재(식료품·주거·보건), 고소득층은 선택재(교육·교통·여가)에 집중 — 우선순위가 정반대
  - 체감물가 역진성: 누적 체감물가 분위1 20.38% vs 분위5 19.17% (**+1.21%p**), 부트스트랩 CI가 0을 배제하고 파셰·피셔에서도 부호 일관 → 견고
  - 격차 '고착': 구조격차지수가 2024Q1 정점(0.303) 후 회귀했으나 변화량 CI가 0을 포함 → 확대도 축소도 아닌 고착
  - 후생 부담: 저소득층 식료품·주거 부담이 고소득층의 약 1.8–2.1배(엥겔·슈바베)
  - 예측·해석: RandomForest가 단순 베이스라인 대비 약 38% 개선, SHAP 분해 결과 '항목 → 소득분위' 순으로 예측 지배
  - 정책 신호등: 데이터 기반 임계치 기준 2024Q1만 세 지표 동시 적색 → 격차가 동시다발적으로 악화된 유일 국면

- **분석의 한계:** 반복 횡단면 자료로 동일 가구 추적 불가(인과가 아닌 기술적 진단), CPI가 전 분위 공통이라 가격탄력성 식별이 약함(AIDS 확장 여지), 소득 기준 분위라 자산·부채 차원 미반영, 체감물가를 12대 항목 수준에서만 차별화해 역진성은 하한 추정
